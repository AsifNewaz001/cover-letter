## Quick Start

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/cover-letter.git
   cd cover-letter
   ```

2. Open `index.html` in your browser:
   ```bash
   open index.html
   ```

3. Download as PDF:
   - Click the "Download PDF" button
   - Set margins to **None**
   - Enable **Background graphics**
   - Set paper size to **A4**
   - Save as PDF

## Customization

Simply edit `index.html` to customize your cover letter:

- **Personal Information**: Update name, title, and contact details (lines 288-295)
- **Date & Location**: Update date and location (lines 298-301)
- **Recipient Details**: Update hiring manager, team, and company name (lines 304-308)
- **Letter Body**: Customize the content of your letter (lines 314-342)
- **Colors**: Change the teal accent color (`--accent-teal: #07e6c4`) in the CSS

## Letter Structure

The template includes:
- **Header**: Name, professional title, and contact information
- **Date & Location**: Current date and your location
- **Recipient**: Hiring manager details and company information
- **Salutation**: Formal greeting
- **Body Paragraphs**:
  - Opening (express interest and background)
  - Current role and achievements
  - Previous experience
  - Skills and alignment
  - Closing and call to action
- **Signature**: Professional sign-off

## Tips for Writing

- Keep paragraphs concise and focused (3-5 sentences each)
- Lead with your strongest qualifications
- Show specific knowledge about the company
- Quantify achievements where possible
- End with a clear call to action
